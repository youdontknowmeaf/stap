# STAP (**s**imple **t**erminal **a**udio **p**layer)
Terminal audio player supporting mp3, wav and flac. Made in C with miniaudio.

# Install
Compile steps:
  1. Install ``libminiaudio-dev`` using your package manager
  2. Run the make file

Install:
  1. Compile the code
  2. ``sudo mv stap /bin/`` (or ``/usr/bin/``)

# Usage
``stap path/to/file.mp3``
Then press enter to stop playing the file.
